## DSA Assessment 2

#### Problem 1 

Rearrange the runtime complexities in the correct order from faster runtime to slowest 

O(1) -> O(n!) -> O(n ^ 2) -> O(n) -> O(log n) -> O(n log n) -> O(2 ^ n)

<details> 
  <summary>Solution</summary> 
  
  O(1) -> O(log n) -> O(n) -> O(n log n) -> O(2 ^ n) -> O(n!)
  
</details>
